# Meal Tracker

#### 06/03/2016

#### By Ming Wen

## Description

A website built using the Angular 2 framework, allowing the the user to display logged meals.


## Setup/Installation Requirements

- `$ git clone https://github.com/themingfu/javascript_angular_mealtracker`
- `$ npm install`
- `$ bower install`
- `$ gulp build`
- `$ gulp serve`

## Technologies Used
